# Sh1zok's Figura Scripts
This is a repository with my scripts for [Figura](https://github.com/FiguraMC/Figura), a [Minecraft Java](https://www.minecraft.net/en-us/about-minecraft) client [mod](https://modrinth.com) that allows you to extensively customize your player model and have other players see your avatar without requiring any server mods.
